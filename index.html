Sobre

O Alpine Firewall Pro é um sistema completo de bloqueio de conteúdo baseado em Docker + KVM, com painel web Laravel para gerenciamento visual de regras de firewall, logs de auditoria, API REST e suporte multi-tenant.

🚀 Status: Painel admin ativo, CRUD de regras, logs de bloqueio, API REST, documentação embarcada. Login: admin@firewallpro.com / admin123

📊 Painel Admin

Dashboard com estatísticas, regras ativas, logs recentes e gráficos

📝 Regras de Firewall

CRUD completo com toggle ativar/desativar, preview de iptables

📋 Logs de Bloqueio

Histórico com data, IP, domínio, chain e ação

🌐 API REST

Endpoints para gerenciar regras, consultar logs e estatísticas

👥 Multi-tenant

Usuários independentes com suas próprias regras e logs

🔒 LGPD

Logs de auditoria, criptografia em trânsito, relatórios de compliance

Início Rápido

Local

# Clone
git clone https://github.com/TyagoAlves/firewall-pro.git
cd firewall-pro

# Configure o ambiente
cp .env.example .env
php artisan key:generate

# Instale dependências
composer install
npm ci && npm run build

# Banco de dados e seed
php artisan migrate --seed

# Inicie o servidor
php artisan serve --host=0.0.0.0 --port=8000

Acesse http://localhost:8000 — login: admin@firewallpro.com / admin123

Docker

docker compose up -d

Deploy em Produção

# Servidor Ubuntu/Debian
sudo apt install -y nginx php8.3-fpm php8.3-mbstring php8.3-xml \
  php8.3-curl php8.3-sqlite3 php8.3-gd php8.3-zip unzip composer

# Clone e configure
cd /var/www
git clone https://github.com/TyagoAlves/firewall-pro.git
cd firewall-pro
cp .env.example .env
php artisan key:generate
composer install --no-dev
php artisan migrate --seed
php artisan config:cache
php artisan route:cache
php artisan view:cache

Configure o Nginx com SSL para apontar para a pasta public/.

API REST

A API permite gerenciar regras, consultar logs e estatísticas programaticamente.

EndpointMétodoDescrição
/api/rulesGETLista regras de firewall
/api/rulesPOSTCria nova regra
/api/rules/{id}PUTAtualiza regra
/api/rules/{id}DELETERemove regra
/api/logsGETConsulta logs de bloqueio
/api/statsGETEstatísticas do firewall
